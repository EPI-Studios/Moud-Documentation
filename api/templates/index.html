Why Moud?

Four reasons Moud change the way you build.

01

TypeScript Everywhere

One language for server and client. No more context switching.

02

Instant Hot Reload

No restarts. Save, test, and keep your worlds alive while code updates in place.

03

Advanced Rendering

Shaders, lighting, and post-processing live in one unified pipeline.

04

Reactive State Sync

Reliable shared state keeps every player in sync automatically.

Fluent API

Forget the boilerplate. Moud provides an event-driven API that makes creating stuff intuitive.

  • ✓ Fully typed Events
  • ✓ Zero Java knowledge required (maybe if you want to understand deeper by reading the code base)
hello.ts 
events.on("player.chat", (player: any, message: string) => {
    if (message?.trim().toLowerCase() === "hello") {
        player.sendMessage("§aHello from Moud!");
    }
});
