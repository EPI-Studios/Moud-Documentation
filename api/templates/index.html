Get Started in Minutes

Install the CLI and launch your first Moud project with just a few commands.

1. Install the CLI

Get the Moud command-line tool from npm.

npm install -g @epi-studio/moud-cli

2. Create a Project

Scaffold a new project in a new directory.

moud create my-game

3. Run the Dev Server

Start the server with hot-reload enabled.

cd my-game && moud dev
